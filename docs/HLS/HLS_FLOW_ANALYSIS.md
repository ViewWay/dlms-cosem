========================================================================
关键参数解析
========================================================================

1. Wrapper Header 格式:
   - Bytes 0-1:   Version (0x0001)
   - Bytes 2-5:   Source WPort (16=client, 1=meter)
   - Bytes 6-9:   Destination WPort
   - Bytes 10-13: Length
   - Bytes 14+:   APDU Data

2. AES-GCM 加密参数:
   - Key Size: 128 bits (16 bytes)
   - IV Size: 12 bytes
   - Tag Size: 12 bytes
   - AAD: 0x30 + AKEY (用于认证)

3. 帧计数器管理:
   - 初始读取: 0x4B35 (19253)
   - 后续递增: 每次加密操作 +1
   - 同步机制: 请求和响应计数器独立维护

4. GMAC 认证:
   - Security Setup IC: 0x000F
   - 方法 1: 双向认证
   - 生成 Stoken 用于后续通信认证
